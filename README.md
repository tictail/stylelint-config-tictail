# Tictail stylelint config

This is the stylelint config we use at Tictail.

## Install
```
npm i eslint-stylelint-tictail -D
```

## Usage
Add the following to your .stylelintrc:
```
{
  "extends": "stylelint-config-tictail"
}
```
